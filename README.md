1. Foundations
   
Tokenization: Splitting text.
Stemming/Lemmatization: Finding root words.
Stop Words: Removing "filler" words.
POS Tagging: Identifying nouns, verbs, etc.


2. Vectorization (Text to Math)
Bag of Words / TF-IDF: Frequency-based math.
Word Embeddings: Context-based math (Word2Vec, GloVe).


3. Key Applications
Sentiment Analysis: Emotional tone.
NER: Identifying names/dates.
Classification: Spam vs. Not Spam.
Summarization: Shorter versions of text.
Translation: Language to language.


4. Modern Models
RNN / LSTM: Sequential logic.
Transformers: Attention-based logic, self attention
LLMs: GPT, BERT, Llama.
